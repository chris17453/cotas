## DDF (Data Dictionary Files)

- Addsum provides DDF Creator tools for TAS data
- DDF files allow external ODBC/SQL access to Btrieve/Pervasive data
- Available for TAS 4.0, 5.0, 5.1, 6.x, 7i
- Third party tools (Crystal Reports, Access, etc.) can access data via DDFs
- CoTAS uses a similar DDF concept (.int files) for its SQL database mapping
