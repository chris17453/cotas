# DELF()

## Summary
PURPOSE
This function will attempt to delete a file. If the deletion is successful, a logical .T. will be returned.

## Signature
```
PARTS
1
```

## Details
f/c/e
The name of the file to be deleted. This must include the path and extension in standard
