# WRAPS(1)

| | |
|---|---|
| **Category** | Function |
| **Name** | `WRAPS` |
| **Returns** | `I` |

## Purpose

This function returns the size of a line from a field that has been previously set up with the WRAP command.

## Parts

| # | Type | Description |
|---|------|-------------|
| 1 | `f/c/e` | The line number. |

## Return Type

I
