# MNTH(1)

| | |
|---|---|
| **Category** | Function |
| **Name** | `MNTH` |
| **Returns** | `I` |

## Purpose

This function will return the month of a date as a number.

## Parts

| # | Type | Description |
|---|------|-------------|
| 1 | `f/c/e` | The date value to use, must be of D type. |

## Return Type

I The number range returned is from 1 to 12, January thru December.
