# DTOR(1)

| | |
|---|---|
| **Category** | Function |
| **Name** | `DTOR` |
| **Returns** | `R` |

## Purpose

This function will convert a D type field to an R type field.

## Parts

| # | Type | Description |
|---|------|-------------|
| 1 | `f/c/e` | The D type field to be converted. The original field is not changed. |

## Return Type

R The original field is not changed.
