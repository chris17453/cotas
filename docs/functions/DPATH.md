# DPATH()

| | |
|---|---|
| **Category** | Function |
| **Name** | `DPATH` |
| **Returns** | `A` |

## Purpose

This function will return the default dictionary path value as set in the TAS50.OVL file or by the CHANGE DICTIONARY PATH command.

*No parameters.*

## Return Type

A

## Example

```tas
? dpath()
C:\ACCTG\
```
