# ATAN(1)

| | |
|---|---|
| **Category** | Function |
| **Name** | `ATAN` |
| **Returns** | `N` |

## Purpose

This function returns the arctangent of the N type field specified.

## Parts

| # | Type | Description |
|---|------|-------------|
| 1 | `f/c/e` | The N value. |

## Return Type

N The result will be from -PI/2 to PI/2 in radians.

## Example

```tas
? atan(-0.539)
-0.49435871
```
