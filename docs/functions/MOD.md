# MOD(1,2)

| | |
|---|---|
| **Category** | Function |
| **Name** | `MOD` |
| **Returns** | `N` |

## Purpose

This function returns the division remainder of two specified numbers.

## Parts

| # | Type | Description |
|---|------|-------------|
| 1 | `f/c/e` | The dividend, where quotient = dividend / divisor. This must be a type N value. |
| 2 | `f/c/e` | The divisor. This must be a type N value. |

## Return Type

N
