# CTOT(1)

| | |
|---|---|
| **Category** | Function |
| **Name** | `CTOT` |
| **Returns** | `T` |

## Purpose

This function converts an A (character string) type field to a T (time) type field.

## Parts

| # | Type | Description |
|---|------|-------------|
| 1 | `f/c/e` | The A type field that is to be converted. |

## Return Type

T
