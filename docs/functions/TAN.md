# TAN(1)

| | |
|---|---|
| **Category** | Function |
| **Name** | `TAN` |
| **Returns** | `N` |

## Purpose

This function returns the tangent of the N type field specified.

## Parts

| # | Type | Description |
|---|------|-------------|
| 1 | `f/c/e` | The N value. |

## Return Type

N The return value is in radians.

## Example

```tas
? tan(1.539)
31.43957419
```
