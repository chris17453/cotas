# YEAR(1)

| | |
|---|---|
| **Category** | Function |
| **Name** | `YEAR` |
| **Returns** | `A` |

## Purpose

This function returns the year of a date value (type D) as a string (type A).

## Parts

| # | Type | Description |
|---|------|-------------|
| 1 | `f/c/e` | The date value to use. |

## Return Type

A This function returns a 4 character string.

## Example

```tas
? year(date())
1994
```
