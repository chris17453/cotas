# RTOD(1)

| | |
|---|---|
| **Category** | Function |
| **Name** | `RTOD` |
| **Returns** | `D` |

## Purpose

This function will convert a value in an R type field to a date (D type) value.

## Parts

| # | Type | Description |
|---|------|-------------|
| 1 | `f/c/e` | The R type value to convert. The original field is not changed. |

## Return Type

D
