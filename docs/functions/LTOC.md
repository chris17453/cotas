# LTOC(1)

| | |
|---|---|
| **Category** | Function |
| **Name** | `LTOC` |
| **Returns** | `A` |

## Purpose

This function converts a L type value to an A type value.

## Parts

| # | Type | Description |
|---|------|-------------|
| 1 | `f/c/e` | The logical value to convert. |

## Return Type

A
