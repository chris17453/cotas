# TTOR(1)

| | |
|---|---|
| **Category** | Function |
| **Name** | `TTOR` |
| **Returns** | `R` |

## Purpose

This function will convert a time (type T) value to a record/long integer (type R) value.

## Parts

| # | Type | Description |
|---|------|-------------|
| 1 | `f/c/e` | The time value to use. The original field is left unchanged. |

## Return Type

R

## Sample Program

`TIMETEST.SRC`
