# WRAPO(1)

| | |
|---|---|
| **Category** | Function |
| **Name** | `WRAPO` |
| **Returns** | `I` |

## Purpose

This function will return the offset of the beginning of a line from a field that has been previously set up with the WRAP command.

## Parts

| # | Type | Description |
|---|------|-------------|
| 1 | `f/c/e` | The line number. |

## Return Type

I The first character of any field is position 1.
