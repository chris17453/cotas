# CTOL(1)

| | |
|---|---|
| **Category** | Function |
| **Name** | `CTOL` |
| **Returns** | `L` |

## Purpose

This function converts an A (character string) type field to a L (logical) type field.

## Parts

| # | Type | Description |
|---|------|-------------|
| 1 | `f/c/e` | The A type field that is to be converted. |

## Return Type

L

## Example

```tas
? ctol(‘.Y.’)
.T.
```
