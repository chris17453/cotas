# MEM()

| | |
|---|---|
| **Category** | Function |
| **Name** | `MEM` |
| **Returns** | `I` |

## Purpose

This function returns the number of kbytes (each kbyte is 1024 bytes) of memory not currently being used and available.

*No parameters.*

## Return Type

I

## Example

```tas
? mem()
264

&& 264k or 270,336 bytes
```
