# ASC(1)

| | |
|---|---|
| **Category** | Function |
| **Name** | `ASC` |
| **Returns** | `I` |

## Purpose

This function will return the ASCII value of the first character of a field.

## Parts

| # | Type | Description |
|---|------|-------------|
| 1 | `f/c/e` | The field to be used. The function will use only the first character of the field which must be of A type. |

## Return Type

I The ASCII value.
