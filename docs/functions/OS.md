# OS()

| | |
|---|---|
| **Category** | Function |
| **Name** | `OS` |
| **Returns** | `A` |

## Purpose

This function returns the operating system name and version number.

*No parameters.*

## Return Type

A The size of the returned value depends on the name of the operating system and the version number.

## Example

```tas
? OS()
DOS 5.1
```
