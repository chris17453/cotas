# TTOF(1)

| | |
|---|---|
| **Category** | Function |
| **Name** | `TTOF` |
| **Returns** | `N` |

## Purpose

This function will convert a time (type T) value to a numeric (type N) value.

## Parts

| # | Type | Description |
|---|------|-------------|
| 1 | `f/c/e` | The time value to use. The original field is left unchanged. |

## Return Type

N

## Sample Program

`TIMETEST.SRC`
