# COS(1)

| | |
|---|---|
| **Category** | Function |
| **Name** | `COS` |
| **Returns** | `N` |

## Purpose

This function returns the cosine of the N type field specified.

## Parts

| # | Type | Description |
|---|------|-------------|
| 1 | `f/c/e` | The N value. |

## Return Type

N The return value is in radians.

## Comments

The value specified must be between -1 and 1.

## Example

```tas
? cos(1.539)
0.03179097
```
