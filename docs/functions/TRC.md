# TRC(1)

| | |
|---|---|
| **Category** | Function |
| **Name** | `TRC` |
| **Returns** | `R` |

## Purpose

This function returns the number of records in a specified file.

## Parts

| # | Type | Description |
|---|------|-------------|
| 1 | `f/c/e` | The file_number of the file to use. |

## Return Type

R
