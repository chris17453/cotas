# FTOT(1)

| | |
|---|---|
| **Category** | Function |
| **Name** | `FTOT` |
| **Returns** | `T` |

## Purpose

This function will convert an N type field to a T (time) field.

## Parts

| # | Type | Description |
|---|------|-------------|
| 1 | `f/c/e` | The N type field value to be converted to a T field. The original field is unchanged. |

## Return Type

T
