# DSPCE()

| | |
|---|---|
| **Category** | Function |
| **Name** | `DSPCE` |
| **Returns** | `R` |

## Purpose

This function will return the amount of unused disk space on the current ‘default’ drive.

*No parameters.*

## Return Type

R The value returned is in number of bytes. If there are 5 Mbytes remaining the returned value would be: 5242880
