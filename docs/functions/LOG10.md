# LOG10(1)

| | |
|---|---|
| **Category** | Function |
| **Name** | `LOG10` |
| **Returns** | `N` |

## Purpose

This function will return the log base 10 of the numeric value specified.

## Parts

| # | Type | Description |
|---|------|-------------|
| 1 | `f/c/e` | The numeric value to use. |

## Return Type

N

## Example

```tas
? log10(9000.000)
3.954
```
