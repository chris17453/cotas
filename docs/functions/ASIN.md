# ASIN(1)

| | |
|---|---|
| **Category** | Function |
| **Name** | `ASIN` |
| **Returns** | `N` |

## Purpose

This function returns the arcsine of the N type field specified.

## Parts

| # | Type | Description |
|---|------|-------------|
| 1 | `f/c/e` | The N value. |

## Return Type

N The result will be from -PI/2 to PI/2 in radians.

## Comments

The value specified must be between -1 and 1.
