# CTOD(1)

| | |
|---|---|
| **Category** | Function |
| **Name** | `CTOD` |
| **Returns** | `D` |

## Purpose

This function converts an A (character string) type field to a D (date) type field.

## Parts

| # | Type | Description |
|---|------|-------------|
| 1 | `f/c/e` | The A type field that is to be converted. |

## Return Type

D
