# SIGN(1)

| | |
|---|---|
| **Category** | Function |
| **Name** | `SIGN` |
| **Returns** | `N` |

## Purpose

This function will return an N type value indicating the sign of the field value specified.

## Parts

| # | Type | Description |
|---|------|-------------|
| 1 | `f/c/e` | N type field to check. |

## Return Type

N
