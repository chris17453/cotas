# GET_ELEM_NUM()

| | |
|---|---|
| **Category** | Function |
| **Name** | `GET_ELEM_NUM` |
| **Platform** | Windows Only |
| **Returns** | `R` |

## Purpose

This function returns the element number for the last or current entry field.

*No parameters.*

## Return Type

R Returns the element number for the last or current entry field.
