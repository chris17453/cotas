# PRINTER_NAME()

| | |
|---|---|
| **Category** | Function |
| **Name** | `PRINTER_NAME` |
| **Platform** | Windows Only |
| **Returns** | `A` |

## Purpose

Every printer in Windows has a distinct name. This function will return the name of the currently chosen printer..

*No parameters.*

## Return Type

A The name of the currently active or default printer.
