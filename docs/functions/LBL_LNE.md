# LBL_LNE(1)

| | |
|---|---|
| **Category** | Function |
| **Name** | `LBL_LNE` |
| **Returns** | `I` |

## Purpose

This function will return the internal (program) location for a given label number.

## Parts

| # | Type | Description |
|---|------|-------------|
| 1 | `f/c/e` | The number of the label location to be found. |

## Return Type

I The internal location of the label.
