# IFCR()

| | |
|---|---|
| **Category** | Function |
| **Name** | `IFCR` |
| **Returns** | `L` |

## Purpose

This function checks if the last character entered by the user was the Enter key (just a CR - carriage
return).

*No parameters.*

## Return Type

L If the last key pressed was the Enter key then the function will return .T., otherwise .F.
