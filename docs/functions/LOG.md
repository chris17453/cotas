# LOG(1)

| | |
|---|---|
| **Category** | Function |
| **Name** | `LOG` |
| **Returns** | `N` |

## Purpose

This function will return the log base e of the numeric value specified.

## Parts

| # | Type | Description |
|---|------|-------------|
| 1 | `f/c/e` | The numeric value to use. |

## Return Type

N

## Example

```tas
? log(9000.000)
9.105
```
