# DOW(1)

| | |
|---|---|
| **Category** | Function |
| **Name** | `DOW` |
| **Returns** | `I` |

## Purpose

This function will return the day of the week as a number.

## Parts

| # | Type | Description |
|---|------|-------------|
| 1 | `f/c/e` | The date value to use, must be of D type. |

## Return Type

I The number range returned is from 1 thru 7, Sunday thru Saturday.
