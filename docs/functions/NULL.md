# NULL(1)

| | |
|---|---|
| **Category** | Function |
| **Name** | `NULL` |
| **Returns** | `L` |

## Purpose

This function checks for a null pointer value. Will work with both P and F type fields.

## Parts

| # | Type | Description |
|---|------|-------------|
| 1 | `f/c/e` | Pointer value to check. |

## Return Type

L If null the function will return .T., otherwise .F.
