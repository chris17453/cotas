# EXP(1)

| | |
|---|---|
| **Category** | Function |
| **Name** | `EXP` |
| **Returns** | `N` |

## Purpose

This function returns the exponential of the field given.

## Parts

| # | Type | Description |
|---|------|-------------|
| 1 | `f/c/e` | The N type field to be used in the calculation. |

## Return Type

N

## Comments

The function returns ex, with x being the field value given.

## Example

```tas
? logtest.sr c
```

## Sample Program

`LOGTEST.SRC`
