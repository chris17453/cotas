# TIME()

| | |
|---|---|
| **Category** | Function |
| **Name** | `TIME` |
| **Returns** | `T` |

## Purpose

This function will return the current system time.

*No parameters.*

## Return Type

T

## Comments

This is the same time that is returned if the user enters TIME at the DOS prompt.

## Sample Program

`TIMETEST.SRC`
