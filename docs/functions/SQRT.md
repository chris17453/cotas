# SQRT(1)

| | |
|---|---|
| **Category** | Function |
| **Name** | `SQRT` |
| **Returns** | `N` |

## Purpose

This function will return the square root of the value specified.

## Parts

| # | Type | Description |
|---|------|-------------|
| 1 | `f/c/e` | The value to use. Must be of N type. |

## Return Type

N

## Example

```tas
? sqrt(4)
2
? sqrt(99.325)
9.966
```
