# SIN(1)

| | |
|---|---|
| **Category** | Function |
| **Name** | `SIN` |
| **Returns** | `N` |

## Purpose

This function returns the sine of the N type field specified.

## Parts

| # | Type | Description |
|---|------|-------------|
| 1 | `f/c/e` | The N value. |

## Return Type

N The value returned is in radians.
