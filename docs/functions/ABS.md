# ABS(1)

| | |
|---|---|
| **Category** | Function |
| **Name** | `ABS` |
| **Returns** | `N` |

## Purpose

This function returns the absolute value of the N type field specified.

## Parts

| # | Type | Description |
|---|------|-------------|
| 1 | `f/c/e` | The N type value. |

## Return Type

N

## Example

```tas
? abs(-535.25)
535.25
```
