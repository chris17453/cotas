# ELSE

| | |
|---|---|
| **Category** | Command |

## Description

This is a process control command, i.e., it will control whether a command, or group of commands will be executed. This is one portion of a complete command structure.

## Syntax

```text
ELSE
```

## Comments

For more information on the different structured programming commands, and the ELSE command in particular, please see Chapter 7, Structured Programming Commands.

## Program Editor

`Prg Control -> If -> Else`
