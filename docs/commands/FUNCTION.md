# FUNCTION

| | |
|---|---|
| **Category** | Command |

## Description

This is the command that defines a User Defined Function, or UDF. A UDF may be used in any situation that would allow a regular (standard) function. Some options in some commands require that a UDF be used.

## Syntax

```text
FUNC function_name function_parameters
```

## Parameters

- **`function_name`** 路 `sac` 路 *Required*

  This is a special alpha constant. It is the name you give to this function. It may be up to 14 characters long. It must not be the same as any line label used anywhere else in the program.

- **`function_parameters`** 路 `sac` 路 *Required*
