# READ RECORD

| | |
|---|---|
| **Category** | Command |

## Description

This is a TAS Professional 3.0 command here for compatibility. The preferred method is to use the OPENV/FINDV commands.

## Syntax

```text
RDREC err_label
```

## Parameters

- **`err_label`** · `label` · *Optional*

  This is the label to go to if an error occurs while reading from this non-TAS file.
