# BACKGROUND

| | |
|---|---|
| **Category** | Command |
| **Abbreviation** | `BKG` |
| **Platform** | DOS only |

## Description

This command will change the screen output color to the value set as the highlight color in the COLOR command.

## Syntax

```text
BKG
No options
```

## Program Editor

`User interface -> Color Control -> Background`
