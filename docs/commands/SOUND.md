# SOUND

| | |
|---|---|
| **Category** | Command |

## Description

This command will use an array of notes and beats (number of clock ticks to keep sound on) to make noise come out of the PC speaker.

## Syntax

```text
SOUND note_array BEAT beat_array NUM num_of_notes FILE wav_file
```

## Parameters

- **`note_array`** · `sac` · *Required*

  An array of values that represent notes. Each note is the number 1,193,180/frequency desired. Must be an I type array.

- **`beat_array`** · `sac` · *Required*

  

- **`num_of_notes`** · `n` · *Required*

  

- **`wav_file`** · `FILE` · *Required*
