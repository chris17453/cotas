# CHAINR

| | |
|---|---|
| **Category** | Command |

## Description

This program will CHAIN to a program already specified as a RAP program.

## Syntax

```text
CHAINR rap_number
```

## Parameters

- **`rap_number`** · `f/c/e` · *Required*

  The RAP buffer number. This is the same value as the NUMBER option as set in the RAP (Run Anytime Program) command.
