# LOOP

| | |
|---|---|
| **Category** | Command |

## Description

This is a process control command, i.e., it will loop back to the beginning in a WHILE/ENDW loop.
This is one part of a complete command structure.
LOOP

## Syntax

```text
LOOP
```

## Comments

For more information on the different structured programming commands, and the LOOP command in particular, please see Chapter 7, Structured Programming Commands.

## Program Editor

`Prg control -> While -> Loop`
