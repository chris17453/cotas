# EXECUTE PROGRAM

| | |
|---|---|
| **Category** | Command |

## Description

You would use this command to execute a non-TAS Professional 5.1 program.

## Syntax

```text
EXEC program_name WITH with
```

## Parameters

- **`program_name`** · `f/c/e` · *Required*

  This is the name of the program to be run. This must include any path if applicable.
