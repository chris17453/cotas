# SCRN

| | |
|---|---|
| **Category** | Command |

## Description

This command will allow you to lock, unlock, refresh, and reset the screen.

## Syntax

```text
SCRN what_to_do
```

## Parameters

- **`what_to_do`** · `f/c/e` · *Required*

  L - lock the screen; all display to the screen will be stopped until the unlock or refresh option is run. U - unlock the screen; DO NOT refresh any screen fields currently displayed. R - refresh any fields currently displayed on the screen. S reset the screen, and eliminate the effect of the current window.

## Program Editor

`User interface -> Screen control -> screen L/u/r/s`
