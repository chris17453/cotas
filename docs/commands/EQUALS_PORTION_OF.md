# EQUALS PORTION OF

| | |
|---|---|
| **Category** | Command |

## Description

This is a TAS Professional 3.0 command here for compatibility. The preferred method is to use the =MID() function.

## Syntax

```text
EQU_MID recv_field FLD parse_fld START start_val NCHR num_chrs MEM_AREA
```

## Parameters

- **`recv_field`** · `fn/v` · *Required*

  The field that is going to receive the portion of the parse_fld.

- **`parse_fld`** · `fn/v` · *Required*

  The field that you want a part of.

- **`START`** · `n` · *Required*

  start_val - The starting position for the portion to extract.

- **`NCHR`** · `n` · *Required*

  num_chrs - The number of characters to extract.

- **`MEM_AREA`** · `mem_area` · *Required*

  MEM_AREA - The memory area.
