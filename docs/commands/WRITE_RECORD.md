# WRITE RECORD

| | |
|---|---|
| **Category** | Command |
| **Platform** | TAS |

## Description

This is a TAS Professional 3.0 command here for compatibility. The preferred method is to use the OPENV/SAVE commands.

## Syntax

```text
WTREC err_label
```

## Parameters

- **`err_label`** · `label` · *Optional*

  err_label - label - Optional - This is the label to go to if an error occurs while writing to this non-TAS file.

## Comments

This is the equivalent of the TAS Professional 3.0 command Write non-TAS Record.

## Program Editor

`PROGRAM EDITOR`
