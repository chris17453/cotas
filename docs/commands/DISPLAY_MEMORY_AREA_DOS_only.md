# DISPLAY MEMORY AREA (DOS only)

| | |
|---|---|
| **Category** | Command |

## Description

This is a TAS Professional 3.0 command here for compatibility. There is no preferred TAS Professional 5.1 command/function.

## Syntax

```text
DISPM mem_area# START start_val OCCURS size_of_element OFST offset_in_element AT
col_row_loc WDT #_chrs_to_prt NLNES #_lines_to_prt
```

## Parameters

- **`mem_area#`** · `n` · *Required*

  

- **`START`** · `n` · *Required*

  

- **`start_val`** · `n` · *Required*

  

- **`OCCURS`** · `n` · *Required*

  

- **`size_of_element`** · `n` · *Required*

  

- **`OFST`** · `n` · *Required*

  

- **`offset_in_element`** · `n` · *Required*

  

- **`col_row_loc`** · `n` · *Required*

  

- **`WDT`** · `n` · *Required*

  

- **`#_chrs_to_prt`** · `n` · *Required*

  

- **`NLNES`** · `n` · *Required*

  

- **`#_lines_to_prt`** · `n` · *Required*
