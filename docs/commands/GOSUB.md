# GOSUB

| | |
|---|---|
| **Category** | Command |
| **Platform** | TAS Professional 5.1 |

## Description

This command will temporarily transfer control to a subroutine. The first line of that routine must be a standard line label.

## Syntax

```text
GOSUB gosub_label
```

## Parameters

- **`gosub_label`** · `label` · *Required*

  Transfer program control to the line label.
