# NO RESTART

| | |
|---|---|
| **Category** | Command |
| **Platform** | TAS Professional 5.1 |

## Description

Normally in the LISTF (List File) commands when the program returns to the command from an enter_udf option the record just accessed becomes the active record at the top of the list. Use this command to keep this from happening.

## Syntax

```text
NORSTRT
```

## Program Editor

`User interface -> Lists -> no reStart`
