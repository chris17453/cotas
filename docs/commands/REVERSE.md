# REVERSE

| | |
|---|---|
| **Category** | Command |
| **Platform** | DOS only |

## Description

This will change the display color to the color you have set as reverse.

## Syntax

```text
REV
```

## Program Editor

`User interface -> Color control -> Reverse`
